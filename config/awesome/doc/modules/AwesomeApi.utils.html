mergeTables (source, target)
Recursively merge two tables. Recursively merge the source table into the target table. The target fields will be overwritten by the source fields, unless the field is a table. In such case, those fields will be merged.

Parameters:

  • source {table} The table from which the fields will be taken.
  • target {table} The table that will receive the fields.

Returns:

    {table} A new table.
